How to use Git and Github together as one!

  1. Go to the folder you want to create the git repo on
  2. To clone a existing github repo: git clone https://github.com/bauski/hello-world
  3. Once you have cloned the repo you can do whatever you want to. Add, modify, delete files.
  4. Once you are done with your changes and want to see what you've done: git status
  5. If you're satisfied with your changes you can "add" your changes to the commit: git add . you can also just add specific files to commit: git add "index.html"
  6. Once added you can commit using: git commit -m "A new commit message" If you don't set a message then the default text editor will be opened so that you can set one.
  7. Finally you must do a "pull" into the branch that you choose: git push -u origin master
  8. The prompt will ask you for your username and password, and once committed you're done.
